# Learning Path

Instructions for Go tutorial chapters.

## Chapter 01: Hello World (chapters/01-basic.go)

- Package declaration
- Main function
- Printing to console

Run: `go run chapters/01-basic.go`

## Chapter 02: Operators & Symbols (chapters/02-operators.go)

- Arithmetic (+, -, \*, /, %)
- Pointers (&, \*)
- Short declaration (:=)
- Blank identifier (\_)

Run: `go run chapters/02-operators.go`
